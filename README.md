# Local Agents Repository

> ℹ️ **Note Importante sur la Synchronisation GitHub**
>
> Ce dépôt Git regroupe **la totalité** du dossier `Local_Agents`. 
> 
> Cela signifie que vos différents sous-projets (comme **Antidex**, **Antidex_V2**, **Gemini_integration**, etc.) ne possèdent pas de dépôts GitHub séparés. Ils sont tous stockés ensemble dans ce *« monorepo »* unique : le dépôt `KevDepp/Local_Agents`.
> 
> Ainsi, à chaque fois que vous souhaitez mettre en ligne des modifications pour l'un de ces sous-projets, vous devez procéder aux commandes de sauvegarde Git (`add`, `commit` et `push`) **depuis ce dossier `Local_Agents`**. 
> 
> Tout est donc centralisé ici ! 🚀
