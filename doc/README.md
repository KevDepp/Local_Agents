# Local Agents (Codex ↔ Antigravity)

Ce dossier contient des POCs **locaux** (extension VS Code/Antigravity + robot RPA) pour pousser un prompt dans un chat puis appuyer sur **Enter**.

## Contenu
- `prompt-bridge/` : extension VS Code compatible forks (Antigravity) avec un petit serveur HTTP local.
- `rpa-chat/` : Robot Framework + RPA Framework (attended) pour automatiser focus → paste → Enter.
